# HedNet Protocol

**HedNet Protocol** enables users to share their idle bandwidth and computing resources (CPU and GPU power) directly from their browser. Through real-time performance tracking and profile management, users passively contribute to a scalable network and earn rewards.

## Earning Mechanics

1. **Browser Extension:** Install the HedNet Protocol Extension from the Chrome Web Store.
2. **Resource Sharing:** Allow the extension to utilize your idle internet connection and processing power in the background.
3. **Real-time Tracking:** Monitor your contribution and accumulated rewards via the extension's dashboard.

## Key Features

- **Multi-Resource Harvesting:** Uniquely combines both bandwidth and idle compute (CPU/GPU) sharing.
- **Lightweight:** Runs seamlessly within Chrome without needing heavy desktop clients.
- **Passive Growth:** Set it and forget it. Earning happens passively as long as your browser is open.
