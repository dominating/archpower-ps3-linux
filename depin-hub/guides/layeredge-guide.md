# LayerEdge Node Setup Guide

**LayerEdge** is an incentivized DePIN testnet project that brings scalable compute to Bitcoin via light node verification. By running a node, you earn EDGE points for a potential airdrop.

## Requirements
- A PC, Mac, or VPS (Linux/Windows).
- EVM-compatible wallet (MetaMask, Rabby).

## Setup Instructions

1. **Visit the Dashboard**: Go to the [LayerEdge Dashboard](https://dashboard.layeredge.io).
2. **Connect Wallet**: Connect your EVM wallet.
3. **Claim Daily Rewards**: Start by claiming your daily check-in reward.
4. **Start the Node**: Click on the **Start Node** button. Leave the tab open to run the light node in your browser.
5. **Keep it Running**: Ensure you don't close the tab for at least 10 minutes to register the session.

*Pro Tip: For 24/7 uptime without keeping your primary PC on, consider running the browser session on a low-cost VPS.*
