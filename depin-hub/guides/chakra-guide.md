# Chakra Network Setup Guide

**Chakra Network** is an AI-optimized DePIN project on Solana that maps the public web into an open, permissionless database. By using their Scout browser extension, users can share unused internet bandwidth to collect public web traffic and earn rewards.

## Requirements
- A Chromium-based browser (Chrome, Brave, Edge).
- Solana-compatible wallet (Phantom, Solflare).

## Setup Instructions

1. **Install the Extension**: Find and install the Chakra Scout extension from the Chrome Web Store (or official links).
2. **Create/Connect Account**: Open the extension and connect your Solana wallet or register an account.
3. **Keep it Active**: The extension will quietly use minimal idle bandwidth to map data. Ensure your browser is open for passive earning.
4. **Track Earnings**: Check the extension dashboard to see your network contribution and airdrop points.

*Pro Tip: Pair this with a dedicated browser profile if you run multiple DePIN extensions to keep things organized!*
