# OpenLedger Node Guide

**OpenLedger** is a decentralized AI data infrastructure. Early participants who run a node will accumulate testnet points for exclusive token airdrops.

## Requirements
- PC or VPS (Linux/Windows).
- Chrome or Brave browser.

## Setup Instructions

1. **Register**: Visit the OpenLedger portal and create an account.
2. **Install the Extension**: Install the OpenLedger data sharing extension on your browser.
3. **Connect to Datanets**: Opt-in to key datanets like Desci Datanet and IP Datanet.
4. **Accumulate Points**: Leave the extension running in the background. As an early node runner, you accumulate testnet points.
5. **Future Testnets**: Active runners will get priority access to future testnet phases and exclusive airdrops.
