# HedNet Protocol

## Overview
HedNet Protocol is a decentralized physical infrastructure network (DePIN) launched in mid-2025 that enables you to share your idle bandwidth and computing resources directly from your browser. 

Unlike traditional bandwidth sharing, HedNet also taps into idle CPU and GPU power, making it a robust platform for scalable cloud computing and AI workloads. 

## Key Features
*   **Resource Sharing:** Contribute bandwidth, CPU, and GPU power.
*   **Browser-Based:** Easy to run using their official Chrome Extension.
*   **Real-Time Tracking:** Provides built-in dashboards for performance tracking and profile management.
*   **Passive Income:** Earn tokens passively as long as the browser is running and the extension is active.

## How to Get Started
1. Install the HedNet Protocol Extension from the Chrome Web Store.
2. Create an account and sign in via the extension.
3. Keep the extension enabled to passively contribute bandwidth and compute.

*Disclaimer: Running compute-heavy DePIN nodes via browser extensions may affect system performance.*
