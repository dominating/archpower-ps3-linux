# AIOZ Network DePIN Guide

**AIOZ Network** is a comprehensive Web3 infrastructure platform integrating AI compute, data storage, and video streaming. It leverages a global network of community-run nodes to power its decentralized Content Delivery Network (dCDN).

## What is AIOZ DePIN?
By running an AIOZ Node, you contribute your spare storage, processing power (transcoding), and bandwidth to the network. This infrastructure supports global media delivery, AI training, and dApp hosting. In exchange, node operators earn `$AIOZ` tokens.

## How to Set Up an AIOZ Node
1. **Download the Node Client:** Visit the official [AIOZ Network website](https://aioz.network) and download the node software for your OS (Windows, macOS, Linux).
2. **Wallet Setup:** Connect your AIOZ wallet to receive rewards.
3. **Configure Resources:** Allocate the amount of storage and bandwidth you wish to share.
4. **Start the Node:** Keep the node running to continuously process and deliver media files.
