# DAWN Network ($DAWN)

## Overview
DAWN ($DAWN) is a prominent Solana-based DePIN project that turns your idle internet bandwidth into blockchain rewards. It aims to build a community-driven internet infrastructure by utilizing tokenized bandwidth.

## Key Features
*   **Solana Ecosystem:** Fast, low-fee reward distributions powered by Solana.
*   **Validator Extension:** The DAWN Validator Extension allows users to participate directly from their browser, validating bandwidth in real-time.
*   **DAWN Black Box:** Advanced users can opt for DAWN's proprietary hardware (Black Box) for simultaneous, high-efficiency participation.
*   **Proof of Bandwidth:** Uses cryptographic proofs to verify the bandwidth you are actively contributing.

## How to Get Started
1. Install the DAWN Validator Extension on your browser.
2. Link a Solana wallet (like Phantom) to receive $DAWN rewards.
3. Keep the extension active to earn. Advanced users can purchase the Black Box hardware for enhanced yields.
