# Interlink (ITLG) Setup Guide

**Interlink (ITLG)** is a dual-token DePIN project with an emphasis on real-world utility and long-term cryptographic resilience. With an airdrop confirmed for 2026, it rewards users for providing infrastructural support.

## Requirements
- PC or VPS (Windows/Linux/Mac).
- Crypto Wallet.

## Setup Instructions

1. **Register**: Sign up on the Interlink portal.
2. **Download Node Client**: Get the official Interlink node software for your OS.
3. **Configure & Run**: Connect your account via the app interface or CLI token.
4. **Accumulate**: Leave the node running to passively accrue points towards the ITLG airdrop.

*Pro Tip: VPS environments are highly recommended for optimal uptime and maximum reward accumulation prior to the TGE.*
