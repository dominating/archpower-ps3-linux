# Solix DePIN Extension Guide

**Solix Network** is a new DePIN (Decentralized Physical Infrastructure Network) project that allows you to monetize your unused internet bandwidth by running a lightweight browser extension.

## What is Solix?
Solix taps into idle bandwidth and CPU power to support decentralized networking tasks, AI data scraping, and other distributed computing workloads. Users are rewarded with `$SLIX` tokens for their contribution.

## How to Get Started
1. **Install the Extension:** Download the Solix extension from the Chrome Web Store.
2. **Create an Account:** Register using your email or a Web3 wallet.
3. **Run in the Background:** The extension runs securely in the background, utilizing only idle resources.
4. **Earn Rewards:** Accumulate SLIX points which will be converted to tokens during the TGE (Token Generation Event).

## Requirements
- A stable internet connection.
- A Chromium-based browser (Chrome, Brave, Edge).

*Note: Always use a dedicated browser profile for DePIN extensions to maintain optimal security and privacy.*
