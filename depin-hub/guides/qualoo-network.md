# Qualoo Network Guide

## Overview
Qualoo Network is a DePIN project mapping and optimizing global internet quality in real-time. By providing data on network performance, you contribute to a decentralized map of internet connectivity and earn QXT tokens.

## How It Works
Users can participate by running fixed node devices or by using the Qualoo mobile application on their devices. The network compensates contributors with QXT tokens or points that can be converted.

## Earning QXT Tokens
1. **Mobile App:** Install the app to map your local internet coverage and run connectivity tests. Earn points that can be redeemed or exchanged for QXT tokens.
2. **Fixed Nodes:** Operate a fixed node to continuously monitor internet quality from your location and earn direct QXT token rewards.

## Steps to Join
1. Visit the [Qualoo Website](https://www.qualoo.io/).
2. Choose your preferred contribution method (Mobile App or Fixed Node).
3. Set up your node or install the application.
4. Keep the application running to perform tests and earn points/tokens passively.
