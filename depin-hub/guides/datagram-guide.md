# DataGram Node Guide

## What is DataGram?
DataGram is a new decentralized physical infrastructure network (DePIN) designed for decentralized internet traffic routing. It allows users to contribute their idle bandwidth to a distributed routing pool and earn passive rewards.

## Current Phase: Incentivized Alpha Testnet
DataGram is currently in an early **alpha testnet phase**, which historically provides the highest reward multipliers for early adopters.

## Supported Platforms
- Windows
- macOS
- Linux (Headless via CLI/Docker supported)

## How to Get Started
1. **Sign Up**: Navigate to the official DataGram portal and create an account.
2. **Download Node**: Download the appropriate node software for your OS.
3. **Link Device**: Bind the installed node to your account using the unique device token provided on the dashboard.
4. **Leave Running**: Ensure your machine stays online. The node runs quietly in the background, routing minimal traffic and accumulating Alpha Points.

## Yield Profile
- **Current Rewards**: Alpha Testnet Points
- **Conversion**: Points will be converted to the native DataGram token upon mainnet launch and Token Generation Event (TGE).

*Always use a secondary/burner device or VPS for alpha testnet node software when possible.*
